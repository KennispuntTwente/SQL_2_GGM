# K2B

Dit zijn queries voor het vullen van GGM-tabellen, plus enkele extra tabellen,
voor extractie van de BRP uit Centric Key2Burgerzaken. 

Queries zijn op basis van mapping van Maarten-Jan Kampen (gemeente Delft) van Key2Burgerzaken BRP naar GGM.

Deze queries vullen de tabellen die gedefinieerd zijn in de DDL in `ggm_selectie/K2B/BRP_mssql.sql`
(hierin zitten tussen tabellen uit originele GGM DDL ook enkele extra tabelllen die zijn gedefinieerd in de
mapping van Maarten-Jan Kampen).
